
# Kumite Scoreboard


This is actually based on basketball scoreboard, however kumite(karate) have somehow  
same scoring process with basketball, except for warnings and fouls.



## Pointing System


- 3 points -  when player did takedown opponent with a follow-up. or if player did a kick to the head
- 2 points -  when player did a kick to the torso part, (front/back)
- 1 points -  when a player did a punch torso and head.
